  • Notes:

    1. Objects are like dictionaries when you look up a item it gives you a value.
    2. An important concept to get your head around is that objects are assigned by reference. This means that if a variable is assigned to an object that already exists, it will simply point to the exact same space in memory. So any changes made using either reference will affect the same object. (JN2N) - I learned this the hard way. Super important to remember.
